# Tic-Tac-Toe Game (Go)

This project is a Tic-Tac-Toe game implemented in Go. It provides the foundational Go module for a potential command-line or server-based Tic-Tac-Toe application.

## Key Files

*   **`go.mod`**: This file defines the Go module path (`calvinbrown085/tic-tac-toe`) and specifies the minimum Go version required for the project (`1.22.5`). It is also used by the Go toolchain to manage project dependencies.

## How to Run

To run this Go application, you'll need Go installed on your system.

### Prerequisites

*   Go version 1.22.5 or higher

### Steps

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone https://github.com/calvinbrown085/tic-tac-toe.git
    cd tic-tac-toe
    ```
2.  **Run directly:**
    ```bash
    go run .
    ```
3.  **Build and run (optional):**
    ```bash
    go build -o tic-tac-toe
    ./tic-tac-toe
    ```
